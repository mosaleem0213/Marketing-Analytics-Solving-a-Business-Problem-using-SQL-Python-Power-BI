# 📊 Marketing Analytics: Solving a Business Problem using SQL, Python & Power BI

## 🚀 Project Overview

This project focuses on solving a real-world marketing business problem by analyzing customer behavior, engagement, and sentiment. The goal is to derive actionable insights that improve conversion rates and marketing performance.

---

## 🎯 Business Problem

Companies struggle to understand:

* Customer journey behavior
* Engagement performance across campaigns
* Customer sentiment from reviews
* Conversion optimization opportunities

This project addresses these challenges using data analytics.

---

## 🛠️ Tools & Technologies

* **SQL Server** – Data cleaning, transformation & analysis
* **Python** – Sentiment Analysis (NLP)
* **Power BI / Excel** – Dashboard & visualization

---

## 📂 Project Structure

* `SQL Analytics Query.sql` → Data cleaning & transformation queries
* `customer_reviews_enrichment.py` → Sentiment analysis using Python
* `customer_reviews_with_sentiment.xlsx` → Processed dataset
* `Marketing Analytics Dashboards` → Dashboard file
* `Dashboard slide images` → Visual insights

---

## 🔍 Key SQL Work

* Product categorization (Low / Medium / High pricing)
* Customer & geography data enrichment (JOIN operations)
* Data cleaning (removing whitespace issues)
* Engagement data normalization
* Splitting combined metrics (Views & Clicks)
* Duplicate detection using CTE & ROW_NUMBER()
* Handling NULL values using COALESCE

---

## 🧠 Key Analysis Performed

* Customer Journey Analysis
* Conversion Rate Analysis
* Engagement Metrics (Views, Clicks, Likes)
* Sentiment Analysis (Positive / Negative Reviews)
* Campaign Performance Insights
